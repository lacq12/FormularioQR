# Formulario de Contacto con Código QR

Este proyecto crea un formulario de contacto que genera un código QR con la información ingresada.
El QR utiliza formato vCard, lo que permite que, al escanearlo desde un celular, se abra
automáticamente el registro de contacto para ser guardado.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- QRCode.js

## Uso
1. Completar el formulario
2. Presionar "Generar Código QR"
3. Escanear el código desde un celular
4. Guardar el contacto
